О проекте

Простой мессенджер без лишнего

Mayas — приложение для обмена сообщениями, которое разрабатывается в открытую.

Сейчас — доступен клиент для Android, написанный на Kotlin. Это основная платформа проекта на данном этапе.

В планах — клиент для настольных систем, чтобы переписка была доступна и с компьютера.

Проект открытый: исходники, историю изменений и релизы можно смотреть напрямую в репозитории на GitHub — там же и самая точная информация о текущем состоянии дел.
